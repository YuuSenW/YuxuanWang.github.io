- Scientific Paper Writing Agent Project:

This project built a trustworthy scientific paper writing agent targeting cognitive neuroscience. Based on the Qwen3-30B-A3B model, I conducted domain continued pre-training and DPO fine-tuning using 3000 academic papers. To alleviate model hallucinations, I proposed a specialized benchmark and adopted the LangGraph framework. Combined with local RAG retrieval and multi-agent game optimization, the system realizes reliable paper generation with standard citations, with relevant results submitted to AAAI 2027.

- MoEEG General EEG Decoding Model Project:
  
A universal EEG decoding model based on Transformer architecture was designed for brain signal processing. I constructed a joint JEPA and MAE self-supervised pre-training framework, adopting an 8-layer Transformer Encoder integrated with PMoE and domain-customized attention mechanisms. Optimized for inherent EEG data characteristics, the model achieves high-precision signal reconstruction and decoding. It reaches state-of-the-art performance on four diverse EEG classification downstream tasks, and the work has been fully accepted by CogSci 2026.
